# 三土 · 使用说明

一个帮你读英文文章的小工具：自动标出生词、点任意单词看「有温度」的中文讲解、能发音、能收藏、能导出。

---

## 一、第一次打开（重要！）

因为这个 App 没有花钱买苹果的开发者签名，第一次打开 macOS 会拦一下。按下面做就行，只需一次：

1. 把 `三土.app` 拖到「**应用程序**」文件夹。
2. **右键点** `三土.app` → 选「**打开**」。
3. 弹窗里再点一次「**打开**」。

之后就能像普通软件一样双击打开了。

> 如果右键打开仍提示「已损坏 / 无法打开」，打开「**终端**」(Terminal)，粘贴这行回车即可：
> ```
> xattr -dr com.apple.quarantine /Applications/三土.app
> ```

---

## 二、填入你自己的 API Key（讲解功能需要）

生词高亮不需要任何设置，**开箱即用**。但「中文讲解 / 发音讲解」需要一个 AI 的 Key（每人用自己的，免费额度够日常用）。

1. 打开 App，右上角点「**⚙ 设置**」（首次打开会自动弹出）。
2. 选服务商（推荐 **DeepSeek**，便宜量大），把你的 **API Key** 粘进去。
3. 点「**测试**」，显示成功后点「**保存**」。完成！

### 怎么拿 DeepSeek 的 Key
1. 打开 https://platform.deepseek.com → 注册 / 登录。
2. 左侧「API Keys」→「创建 API Key」→ 复制（`sk-` 开头那串）。
3. 回 App 设置里粘贴即可。

> Key 只存在你自己电脑上（`~/Library/Application Support/SanTu/`），不会上传到任何地方。

---

## 三、日常用法

- **导入文章**：粘贴网页链接，或直接粘整段英文 → 回车。
- **看讲解**：点任意单词，右侧弹出讲解；点「🔊」听发音。
- **生词本**：点过的词自动进生词本，可随时回看、跳回原文。
- **书签**：长文读到哪标一下，下次接着读。
- **主题**：右上角换配色。
- **导出**：把带讲解的文章导成 HTML 保存。

读得开心 📖
