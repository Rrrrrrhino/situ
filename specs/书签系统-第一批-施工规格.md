# 施工规格：书签系统 第一批（点空白放线 + 网格书签抽屉 + 活缩略图）

> 给 builder(Sonnet) 冷启动看的施工单。builder 只读本文件即可施工，不必回放历史。
> 视觉定稿依据：`~/Desktop/预览/四土/书签样式对比.html`（用户已逐项批准；本 spec 文字为准，mockup 供观感对照）。

## 1. 背景 / 目标
四土书模式（EPUB 按章分页）目前没有书签。目标：① 阅读页**点任意正文空白处**即在该处放一条「黛蓝墨线·纸签」书签横线（覆盖层，不推动正文）；② 目录抽屉升级为「目录 | 书签」双 tab，书签 tab 是 **MN4 式缩略图网格**，点击跳回原位；③ 书签随书存档持久化，跨字号/窗口尺寸不漂移。文章模式同步换成同一交互与视觉（保留其原有 DOM 持久化机制）。

## 2. 改动范围（严格）
- 允许改：`santu_app/index.html`、`santu_app/app.py`、`specs/_harness/`（可新增/更新验证 harness）、`CHANGELOG`（概述版+详细版）。
- **禁止顺手改**范围外的任何相邻元素/样式/死代码（scope 严格是用户铁律）。生词本、朗读、聚焦等一概不碰。
- 旧的文章模式「武装放置模式」相关代码（`bkArmed/setArmed/bkAdd` 等）**因本次改动而失效的部分**允许删；其余原有死代码不动。

## 3. 关键约束 / 铁律
- 改动前：项目根时间戳备份到同级 `_backups/`（`cp -R`，排除 `_backups` 自身与 `books/`），并 `git add -A && git commit -m "书签系统第一批·施工前存档"`。
- **书模式横线是覆盖层（absolute 定位），绝不 flow 插入**——加/删书签不得引起正文回流、总页数变化。
- **锚定绝不存页码**。存 `{chapter_idx, block_ord}`；页码只在跳转当刻用几何现算。
- 配色/字体（来自项目现有变量，勿新造色）：
  - 线：`1.5px solid rgba(74,107,133,.52)`（黛蓝 `#4a6b85` = 现有 `--h2/--fC`）；hover 加深至 `rgba(74,107,133,.85)`。
  - 纸签 chip：底 `var(--paper)`、边 `1px solid rgba(74,107,133,.45)`、字色 `#4a6b85`、圆角 99px、字号 11–12px、内容 `📑 M月D日`；hover 时内容变 `✕ 移除`。
  - 缩略图内书签线：同色 1px、透明度约 .40（一丝痕迹，不得像遮挡）。
- **没有常驻 ✕**。整条线（含 chip）点击即删除；删除后 toast「已移除书签 · 点此撤销」，6 秒内点 toast 原位找回（同一记录重放）。
- 点空白只认**正文栏内部**（版心内），页边距/工具条点击不响应；不得干扰现有点词讲解（`.w`/`mark.vocab`/`.phrase`）、拖选取词、月牙、播放器。
- 同一分界线重复点击 = 切换删除（不产生重复书签）。
- 单文件行数已超标是历史现状，本次新 JS/CSS 就近归组并加区块注释，不做拆文件重构。

## 4. 设计细节（施工依据）

### 4.1 数据模型（书模式）
书的存档 `data/library/{id}.json` 增加字段：
```json
"bookmarks": [
  {
    "id": "bk_随机串",
    "ch": 3,              // chapter_idx
    "ord": 17,            // 线在本章第 ord 个块【上方】；ord==块总数 表示章末
    "added_at": 1751450000.0,
    "excerpt": "锚点块前 60 字纯文本",
    "snap": [{"t":"p","x":"块文本截断至160字"}, ...]   // 放置时所在页的块快照，≤10块，供缩略图离线渲染
  }
]
```
- 块的定义与现有一致：`.reading-inner` 内 `p,h1,h2,h3,li,blockquote`（叶子块，与 `_soup_to_blocks` 产物对应）。
- `snap` 采集：放置当刻，取锚点页上可见的块（用现有 `_pageOfEl` 判定同页），按序存标签名+文本；同时记 `frac`（线在该页纵向的大致位置 0..1，字段放进记录，缩略图画线用）。

### 4.2 后端（app.py，最小改动）
- `save_session` 书模式分支：接受 `args["bookmarks"]`（list，原样存入 record）。
- `load_archive` 书模式分支：返回 `bookmarks`（缺省 `[]`，旧存档向后兼容）。
- 不新增 API 方法。`py_compile` 必须过。

### 4.3 放置交互（书模式）
- 在 `reading.addEventListener('click')` 现有处理链**尾部**（所有已有分支都未命中时）加书模式分支：
  - 过滤：`bookMode` 为真；`e.target` 不在 `.w, mark.vocab, .phrase, .bookmark-line, button, a, img, .sent 的可点击件` 内；`getSelection()` 为空；点击坐标落在当前列版心矩形内。
  - 锚定：候选块中先按 X 筛同列（`rect.left<=x<=rect.right`），再按 Y 找命中/最近块；点在块上半 → `ord=块下标`，下半 → `ord=块下标+1`。
  - 若该 `(ch, ord)` 已有书签 → 执行删除（切换语义）；否则新增：立即渲染横线 + `setStatus('📑 已放书签')`（1.6s 清）+ `scheduleSave()`。
- 删除（点线/chip）：移除记录与横线，toast「已移除书签 · 点此撤销」，点 status 药丸 6s 内恢复；`scheduleSave()`。

### 4.4 横线渲染（书模式，覆盖层）
- pager inner 内建 `#bkLayer`（absolute 铺满、`pointer-events:none`；每条线自身 `pointer-events:auto`，命中区高约 10px 含 chip）。
- 每条当前章书签：锚块 `el=blocks[ord]`（`ord==len` 用末块底边），线的横向 = 该块所在**列**的左右边界，纵向 y = 块顶 − 块间距一半（`ord==len` 为末块底 + 间距一半）；chip 缀右端。
- 重算时机：`_initPager` 末尾（开章/字号变/窗口变已都汇于此）统一重画。
- 跳转落点闪烁：复用现有 `.bookmark.flash` 思路，给线加 1.1s 微光动画一次。

### 4.5 抽屉「目录 | 书签」双 tab + 缩略图网格
- 现有目录抽屉头部加双 tab：`目录`（原列表不动）/`书签 · N`。样式沿用抽屉现有质感（paper 底、Spectral 标题、accent 下划线 tab，参照生词本大视图 tab 手法）。
- 书签 tab：**两列网格**。每格：
  - 缩略图（约 131×170，圆角 5px，细边框浅投影）：用该书签的 `snap` 块离线渲染 —— 内层 div 宽 340px、`font:9.5px/1.85 Bitter`、`transform:scale(.385)`、absolute 定位防撑宽（教训：内容层必须脱离布局，否则 grid 轨道被 340px 撑爆）；在 `frac` 高度画 1px 黛蓝细线（透明度 .40）。
  - 图下说明：`第 {ch+1} 章 · {M月D日}`；hover 浮出完整 `excerpt`（title 属性即可）；hover 缩略图右上出 ✕ 删除。
  - 点击格子 → 跳转：若 `ch != bookIdx` 先走现有换章路径；渲染完成后 `blocks[ord]` → `_pageOfEl` 算页 → 翻页 → 线闪烁。
- 排序：按 `(ch, ord)` 书内先后。空态文案：「还没有书签。阅读时点一下正文空白处即可放置。」
- 入口：book-bar 目录按钮旁加一枚 `📑` 小按钮（样式同 toc-btn），点击 = 打开抽屉并切到书签 tab。

### 4.6 文章模式（交互统一，机制保持）
- 去掉武装模式：正文空白单击直接放置（沿用现有 `blockAtY/placeBookmarkAt` 的 flow 插入与 HTML 存档机制，**不改成覆盖层**）。
- `.bookmark` 元素重画成 A 样式：黛蓝细线 + 右端日期纸签（放置日期存 `data-date` 属性随 HTML 持久化）；去掉「📑 上次读到这里」长标签与常驻 `bk-x`；点击整条 = 删除（同样 toast+撤销）。
- 顶栏 `bookmarkBtn` 下拉列表保留（跳转/删除照旧），「＋添加书签」按钮改为提示文案「在文中空白处点一下即可放置」。
- `openArchive` 文章分支的「回到上次书签处」逻辑保持可用（类名 `.bookmark` 不变）。
